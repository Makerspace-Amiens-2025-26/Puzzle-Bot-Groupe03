# Puzzle Bot

## Description

Puzzle Bot est une machine automatisée capable de détecter, saisir et assembler les pièces d'un puzzle. Le projet combine plusieurs domaines techniques : conception mécanique, électronique, programmation, vision par ordinateur et fabrication numérique.

## Objectifs
Détecter les pièces d'un puzzle à l'aide d'une caméra.
Identifier leur position et leur orientation.
Déplacer les pièces grâce à un système robotisé.
Assembler automatiquement le puzzle.
## Technologies utilisées

Python

Impression 3D 
CNC / Découpe laser
